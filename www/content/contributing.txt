## Mailing lists

For questions, answers, feedback and other general discussion please
join the <a
href="http://lists.macosforge.org/mailman/listinfo.cgi/dcerpc-devel">dcerpc-devel</a>
mailing list.

Changes to the DCE/RPC source code are posted to the <a
href="http://lists.macosforge.org/mailman/listinfo.cgi/dcerpc-changes">dcerpc-changes</a> mailing list.
